Como usar o Planejamento

Este é um quadro Kanban para organizar suas aulas e atividades.

Básico:

  • Quadros: Crie quadros para matérias ou turmas diferentes.
  • Listas: Use para fases (ex: Planejamento, Em andamento, Concluído).
  • Notas: São suas aulas ou tarefas. Clique para editar. Arraste para mover.

Dicas:

  • Clique no Menu (☰) para exportar ou fazer backup.
  • Use o botão + para novas listas ou notas.
  • No celular, arraste o quadro para o lado para ver mais listas.

Exportação PDF: Agora você pode gerar um PDF completo com seus dados e planejamento para impressão.
